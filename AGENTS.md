# AGENTS.md

## Product
This repo is an internal outbound prospecting and outreach platform.

## Priorities
1. Human review before sending
2. High-quality lead selection
3. Clear provenance and source links
4. Clean CRM-like UX
5. Maintainable code

## Tone
Professional, sharp, premium, concise.

## Engineering rules
- TypeScript everywhere
- Reusable components
- No silent failures
- All workflows idempotent where possible
- Prefer clarity over cleverness

## Data rules
- Use only public, reviewable data sources
- Do not assume contact certainty
- Preserve source URLs
- Deduplicate aggressively

## UX rules
- Desktop-first
- Table-first
- Fast review queue
- Folk-like simplicity

## Default behavior
- Draft before send
- Log all state transitions
- Show assumptions clearly
